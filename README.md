## Getting Started

1. Clone this repository
2. Install dependencies:

```bash
npm install
```

## Running Tests

To run all tests:

```bash
npm test
```

To run a specific test file:

```bash
npm test challenges/daily-YYYY-MM-DD.test.js
```
